# セキュリティポリシー

## 🔒 この拡張機能は安全です

- この拡張機能はユーザーのデータを**一切収集・保存・送信しません**。
- パーミッションは最小限であり、外部通信も行いません。
- 入力フォームやログイン情報へのアクセスも行っておりません。
